## Welcome to *_Jiangsu Xishan Senior HighSchool_* 👋

江苏省锡山高级中学位于江苏省无锡市惠山区。学校始建于清光绪三十三年（1907年）春，前身为无锡私立匡村学校， 1952年12月，学校转为公办，更名为“无锡县中学”；1996年11月，更名为“江苏省锡山高级中学”。
